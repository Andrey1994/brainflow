Key Features


Uniform API for All Devices

Develop device agnostic applications to target more users. More than 10 supported devices.

7 Language Bindings

Deep integration into Python, C++, Java, C# and support for Julia, Matlab and R. Same API across all bindings.

Signal Processing API

In addition to data acquisition API BrainFlow also provides Singal Processing API which can be used independently.
